# ADR-007 - Version Odoo Community: 18.0

Statut: Accepte  
Date: 2026-04-16

## Contexte

Le projet doit equilibrer stabilite applicative, disponibilite des modules et horizon de maintenance.

## Decision

Standardiser l'integration ERP sur `Odoo Community 18.0`.

## Consequences

- base recente pour evolutions fonctionnelles
- image Docker et ecosysteme modules disponibles
- besoin de validation de compatibilite modules a chaque upgrade mineure

## Alternatives considerees

- versions plus anciennes: stabilite historique mais moins de capacites recentes
