# Architecture Decision Records (ADR)

Liste des ADR du projet INA ERP:

- ADR-001 - gestionnaire de paquets `pnpm`
- ADR-002 - monorepo `pnpm workspaces + Turborepo`
- ADR-003 - ORM NestJS `Prisma`
- ADR-004 - framework de test backend `Jest`
- ADR-005 - lint/format `ESLint + Prettier`
- ADR-006 - broker `RabbitMQ`
- ADR-007 - version `Odoo Community 18.0`
- ADR-008 - integration Odoo via `JSON-RPC` + client custom

Format recommande pour la suite:

- statut (`propose`, `accepte`, `deprecie`, `remplace`)
- contexte
- decision
- consequences
- alternatives considerees
